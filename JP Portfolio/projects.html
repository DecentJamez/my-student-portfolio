Projects

A few things I've built

Small, focused tools rather than one giant app, each one taught me something different about how the pieces fit together.

Node.js / API

Live Crypto Price Ticker

A small Node.js script that polls a live price API for Bitcoin's current value every ten seconds using axios, then clears and redraws the terminal with a formatted ticker line and a last updated timestamp. It is a simple project on the surface, but it covers async/await, error handling around a network call, and working with setInterval to keep data fresh without the user doing anything.

Python / Pillow

Image Watermarker

A Python utility built on the Pillow library that stamps a semi transparent text watermark onto the bottom right corner of any image, then exports a clean copy ready to share. It works by creating a fully transparent overlay layer the same size as the source image, drawing the watermark text onto that layer, and compositing the two together before saving the result as a new file.

JavaScript / DOM

Markdown Live Preview

A two panel browser tool, a plain textarea on one side and a live rendered preview on the other. It listens for the textarea's input event and, on every keystroke, runs the typed markdown through the marked.js library and writes the resulting HTML straight into the preview panel, so formatting appears instantly with no separate "render" button needed.
